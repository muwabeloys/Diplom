# 📚 Приложение для изучения иностранных языков

Дипломный проект. Full-stack приложение с системой интервального повторения.

## Технологии
- Frontend: React + Vite
- Backend: Express.js
- База данных: SQLite
- Аутентификация: JWT

## Быстрый старт

```bash
# Установка всех зависимостей
npm run install:all

# Запуск в режиме разработки
npm run dev

Frontend: http://localhost:5173
Backend: http://localhost:5000
